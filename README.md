# Romox

## About Romox
Romox is a program to read out header information of Game Boy and Game Boy Color ROMs written in C++

## Usage

    $ ./romox <game>

## Example
    $ ./romox TETRIS.gb

### Output
    Title: TETRIS
    CGB Flag: Not set
    SGB support: false
    Cartridge type: ROM ONLY
    ROM size: 32 KiB
    RAM size: 0
    Destination: Japan (and possibly overseas)
    Licensee Code: 1
    Mask ROM Version number: 1
